# Getting Started

1. Create `tishift-neon.yaml` from the example config.
2. Run `tishift-neon scan` to assess your Neon/Postgres database.
3. Run `tishift-neon convert` to generate TiDB-compatible DDL.
4. Run `tishift-neon load` to transfer data.
5. Run `tishift-neon check` to validate, then `tishift-neon sync` for CDC.
